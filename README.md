Buenas, este es nuestro repositorio para la segunda entrega de POO del Sistema de Gestión Bibliotecario, desarrollado por Andres Felipe Torres Padilla, Valeria Benitez Cardenas y Juan Pablo Mutis Torres. 
El codigo cumple estrictamente con cada punto de la guia de evaluacion: corregimos el id static de Autor para que no de problemas, 
pasamos Libro a clase abstracta con su metodo abstracto para armar la 
jerarquia de herencia de multiples niveles usando
LibroDigital y LibroAudio,dejamos limpia la agregacion en Categoria y metimos la sobrecarga de constructores en Prestamo para automatizar las fechas.
